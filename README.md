JongDeug.github.io
===================
개발 공부를 기록하는 정적 블로그입니다.
----------------------------------
#### <특징>
Jekyll, Hexo 같은 정적 블로그 생성기를 통해 만든 것이 아니고, 정적 블로그 생성기 자체를 node.js로 만들어서 사용하고 있습니다. 

#### <개발 관련 설명>
1. 개발 언어 
    HTML, CSS, JAVASCRIPT

2. 개발 환경
    Node.js

3. 프레임 워크
    Bootstrap

#### <시작 방법>
git clone 후 npm install
